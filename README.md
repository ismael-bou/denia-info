# denia.info 🇪🇸

![Status](https://img.shields.io/badge/Status-In%20Development-yellow)

The #1 platform for discovering restaurants, bars, hotels, and events in beautiful Denia, Spain. We connect locals and tourists with the best experiences the city has to offer, built on real relationships and localized expertise.

## Project Documentation
All strategic planning and project architecture documents are located in the [docs/](./docs/) directory:

- 📄 [STRATEGY.txt](./docs/STRATEGY.txt): Competitive strategy and our local moat.
- 🗺️ [ROADMAP.txt](./docs/ROADMAP.txt): Week-by-week development plan from MVP to launch.
- 🏗️ [PROJECT.txt](./docs/PROJECT.txt): Complete project vision, tech stack, database structure, and monetization plan.

## Tech Stack
This project is built using a modern, scalable web stack:
- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Backend & Auth:** [Supabase](https://supabase.com/)
- **AI Integration:** Claude AI
