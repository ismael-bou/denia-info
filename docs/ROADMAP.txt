WEEK 1 - FOUNDATION & HOMEPAGE
Goal: Get a live website online
- Day 1: Clone repo + set up Next.js project in Antigravity
- Day 2: Build Homepage (hero, search bar, category cards)
- Day 3: Build Navbar + Footer
- Day 4: Connect Vercel to GitHub (auto-deploy)
- Day 5: Test on mobile, fix design issues
END OF WEEK: denia.info is live with a homepage

WEEK 2 - RESTAURANT PAGES
Goal: Show real restaurants on the site
- Day 1: Build Restaurant Listing page (grid of cards)
- Day 2: Add filters (cuisine, price, area, open now)
- Day 3: Build Restaurant Detail page (menu, photos, hours)
- Day 4: Connect Supabase database
- Day 5: Add 3 fake restaurants to test everything works
END OF WEEK: Restaurants show up on the site from database

WEEK 3 - BOOKING SYSTEM
Goal: Tourists can book a table
- Day 1: Build booking form (name, date, time, guests)
- Day 2: Save bookings to Supabase
- Day 3: Set up SendGrid email confirmations
- Day 4: Test full booking flow start to finish
- Day 5: Fix bugs, test on mobile
END OF WEEK: Someone can book a table and get an email

WEEK 4 - CLAUDE AI CHATBOT
Goal: Smart tourist assistant on the site
- Day 1: Connect Claude API to the project
- Day 2: Build chat widget on homepage
- Day 3: Train chatbot with Denia restaurant data
- Day 4: Add auto-translate for menus (ES/EN/FR/DE/NL)
- Day 5: Test everything, final polish
END OF WEEK: AI chatbot live, menus in 5 languages

WEEK 5 - GO TO MARKET
Goal: Get first real restaurants signed up
- Visit 10 restaurants in Denia personally
- Offer free listing for 6 months
- Add their real menus, photos, info to the site
- Post in Denia Facebook groups
- Contact Denia tourism office
END OF WEEK: 10 real restaurants live on denia.info

MONTH 2-3 - GROWTH
Goal: 30 restaurants, 500 visitors/month
- Add Hotels section
- Add Events calendar
- Add user reviews system
- Start Instagram & TikTok content
- Push notifications for daily specials
- Restaurant owner dashboard

MONTH 4-6 - FIRST REVENUE
Goal: First paying customers
- Launch featured listings (pay to appear at top)
- Premium dashboard with analytics
- Booking commission system
- Expand to Javea & Calpe
- Target: 500-1000 EUR/month revenue

YEAR 2 - EXPANSION
Goal: Full Costa Blanca
- Mobile app iOS & Android
- Expand to Calpe, Altea, Benidorm, Gandia
- Delivery integration
- Loyalty program for users
- White-label for other Spanish towns
